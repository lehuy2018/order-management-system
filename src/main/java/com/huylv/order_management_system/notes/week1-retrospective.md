# Challenges

Một số khó khăn gặp phải:

* Hiểu cách Spring Boot auto configuration hoạt động

* Tổ chức project structure hợp lý

# Key takeaways

Những điều quan trọng rút ra:

1. Không nên expose Entity trực tiếp trong API
2. Service layer nên chứa business logic
3. Controller nên giữ càng đơn giản càng tốt
4. Validation giúp giảm lỗi từ client
